# Getting Started

:::info
This section covers the initial setup required to start developing for the eIsland project, including tool installation, environment configuration, and project bootstrapping.
:::

## Overview

To contribute to eIsland, you need to set up development environments for both the frontend (Electron + React) and backend (Java Spring Boot) components. The plugin system can be developed independently with a minimal toolchain.
