# Guides

:::info
This section provides practical development guides for working with the eIsland codebase, covering daily workflows, plugin creation, and debugging techniques.
:::

## Overview

The eIsland project follows a structured development workflow with clear branching strategies and code review processes. Plugins are first-class citizens in the ecosystem and can be developed, tested, and published independently.
