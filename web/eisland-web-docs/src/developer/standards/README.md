# Standards

:::info
This section documents the coding standards, documentation conventions, and commit message formats enforced across the eIsland project to maintain code quality and consistency.
:::

## Overview

All contributions to eIsland must adhere to the established standards. These cover language-specific style guides, documentation requirements, and version control conventions. Automated linting and CI checks enforce many of these rules.
