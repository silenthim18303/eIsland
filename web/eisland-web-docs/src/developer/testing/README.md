# Testing

:::info
This section documents the testing strategies, frameworks, and practices used across the eIsland project to ensure code quality and reliability.
:::

## Overview

The eIsland project follows a test pyramid approach with unit tests forming the base, integration tests in the middle, and E2E tests at the top. The project maintains **125 test files** with **2068 tests** across all layers.
