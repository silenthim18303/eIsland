# CommandResult

:::info
Result of a media control command execution.
:::

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `success` | `boolean` | Whether the command succeeded |
| `error` | `string \| null` | Error message if failed, `null` on success |
