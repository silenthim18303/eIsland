# getAllDevices

:::info
Returns all visible Bluetooth devices (paired, connected, and discoverable).
:::

## Signature

```typescript
function getAllDevices(): BluetoothDeviceInfo[]
```

## Return Value

Array of [BluetoothDeviceInfo](bluetooth-device-info.md) objects.
