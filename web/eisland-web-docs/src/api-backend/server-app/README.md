# Server App

:::info
The `server-app` module provides application configuration and startup services.
:::

## Overview

The server-app module handles application-level configuration:

- **Application Configuration** — Spring Boot application settings
- **Startup Services** — Application initialization and lifecycle

:::note
This module does not expose any REST API endpoints. It provides internal configuration and startup services for the application.
:::

## Module Structure

| Component | Description |
|---|---|
| Application Config | Spring Boot application configuration |
| Startup Services | Application initialization |
