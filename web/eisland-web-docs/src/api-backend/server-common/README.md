# Server Common

:::info
The `server-common` module provides shared utilities, configurations, and common services used across all modules.
:::

## Overview

The server-common module contains shared components:

- **Utility Classes** — Common utility functions
- **Shared Configurations** — Global configuration settings
- **Common Services** — Shared service implementations

:::note
This module does not expose any REST API endpoints. It provides shared utilities and configurations for other modules.
:::

## Module Structure

| Component | Description |
|---|---|
| Utility Classes | Common utility functions |
| Shared Configurations | Global configuration settings |
| Common Services | Shared service implementations |
