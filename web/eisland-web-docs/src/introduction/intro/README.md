# Introduction

:::info
This section covers the foundational documents for the eIsland project, including the project overview and community guidelines.
:::

## Overview

eIsland is a desktop widget application for Windows that brings the Apple Dynamic Island experience to PC. Built with Electron, it provides a rich set of features including music integration, productivity tools, AI assistant, mini-games, and more.
