> **Release Date:** *`2026-06-20`* **GitHub Repository:** *`https://github.com/JNTMTMTM/eIsland`*

**V26.6.3 新增股票收藏管理与侧边栏交互体系，支持收藏添加、删除、搜索与 K 线周期切换，同步上线 Web 端基础架构与 Simplex 流动背景，同时优化了 K 线图视觉呈现与搜索框交互体验。**

*V26.6.3 introduces a stock favorites management and sidebar interaction system, supporting adding, deleting, searching favorites and K-line period switching, along with the initial Web application architecture and Simplex flow background, while optimizing K-line chart visuals and search box interaction experience.*

## 新功能

- 新增股票收藏管理功能，支持从搜索结果中添加收藏并在侧边栏中查看与管理。
- 新增股票侧边栏，支持收藏列表展示、搜索模式切换与键盘快捷操作。
- 新增股票收藏删除功能，支持单个删除与快捷操作，附带动画过渡效果。
- 新增 K 线图周期切换功能，支持在不同时间周期间快速切换。
- 新增股票搜索关键词记忆与清空功能，输入内容在会话间自动保留。
- 新增股票代码自动规范化与校验，搜索结果解析更可靠。
- 初始化 Web 端基础架构，包含 Vite 构建配置、React 入口与基础组件。
- 新增 Web 端 Simplex 噪声流动全屏背景组件，带来独特的视觉效果。
- 新增 Web 端毛玻璃导航栏组件。
- 优化灵动岛展开状态标签页可见性逻辑，概览标签页始终保持可见。

## 体验优化

- 优化股票搜索面板的键盘与鼠标交互体验，操作更流畅自然。
- 优化股票侧边栏布局与样式，收藏项展示更紧凑美观。
- 优化 K 线图标题样式与图表间距，视觉呈现更简洁。
- 优化 K 线图强调色样式与导航遮罩效果。
- 优化股票侧边栏搜索结果的样式适配，搜索模式下布局更合理。
- 优化股票删除控件的交互体验，删除流程更顺畅。

## 问题修复

- 修复股票搜索结果解析中空值与未定义值检查不严格的问题。
- 修复灵动岛展开状态标签页在不同启动模式下可见性异常的问题。

## 文档更新

- 为多个组件与工具函数补齐版权声明与文件级 JSDoc 注释。
- 补齐股票市场相关功能的中英文多语言翻译。
