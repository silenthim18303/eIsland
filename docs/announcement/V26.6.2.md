> **Release Date:** *`2026-06-17`* **GitHub Repository:** *`https://github.com/JNTMTMTM/eIsland`*

**V26.6.2 新增全屏窗口检测原生插件与灵动岛全屏自动隐藏功能，当检测到全屏应用时灵动岛自动收起，退出全屏后恢复显示，同时补齐了插件的完整测试与文档体系。**

*V26.6.2 introduces a native fullscreen window detection plugin and an auto-hide feature for the Dynamic Island — the island automatically collapses when a fullscreen application is detected and restores when exiting fullscreen, along with a complete test suite and documentation for the plugin.*

## 新功能

- 新增全屏窗口检测原生插件，基于 Win32 API 实现 Windows 平台全屏窗口的实时检测。
- 新增灵动岛全屏自动隐藏功能，当检测到全屏应用运行时灵动岛自动收起，退出全屏后自动恢复显示。

## 体验优化

- 优化公告内容 HTML 渲染安全性，新增 DOMPurify 净化处理。
- 优化媒体协议访问范围，限制仅可访问用户数据壁纸目录。

## 文档更新

- 为全屏检测器插件补齐完整测试套件，覆盖单元测试、轮询测试与冒烟测试。
- 为全屏检测器插件补充类型声明与使用文档。
- 将全屏检测器测试框架迁移至 Vitest，统一项目测试方案。
- 为全屏检测器插件源码与测试文件补齐版权声明与文件级注释。

感谢大家持续反馈与支持。若你在升级后发现新问题，欢迎继续反馈，我们会尽快跟进。

## New Features

- Added a native fullscreen window detection plugin that implements real-time fullscreen window detection on Windows using Win32 API.
- Added Dynamic Island auto-hide for fullscreen applications — the island automatically collapses when a fullscreen app is running and restores when exiting fullscreen.

## Improvements

- Optimized announcement content HTML rendering security with DOMPurify sanitization.
- Restricted media protocol access scope to the user data wallpapers directory only.

## Documentation

- Added a complete test suite for the fullscreen detector plugin, covering unit tests, polling tests, and smoke tests.
- Added type definitions and usage documentation for the fullscreen detector plugin.
- Migrated fullscreen detector tests to Vitest for a unified project testing approach.
- Added copyright headers and file-level comments to fullscreen detector source and test files.

Thank you for your continued feedback and support. If you encounter new issues after upgrading, please continue to report them — we will follow up as soon as possible.
