> **Release Date:** *`2026-07-10`* **GitHub Repository:** *`https://github.com/JNTMTMTM/eIsland`*

**V26.6.4 新增首次启动引导窗口，支持语言选择、主题配置、白名单设置等完整初始化流程，同步上线启动画面视频动画与灵动岛波浪特效，新增歌词翻译与歌词校准功能，并带来 9 个原生 Windows 辅助插件与全面的 API 文档。**

*V26.6.4 introduces a first-launch guide window supporting language selection, theme configuration, whitelist setup and other complete initialization flows, along with splash screen video animations and Dynamic Island wave effects, new lyrics translation and calibration features, and brings 9 native Windows helper plugins with comprehensive API documentation.*

## 新功能

- 新增首次启动引导窗口，支持语言选择、主题配置、更新源选择、白名单设置、SMTC 检测与 GitHub 信息展示等完整初始化流程。
- 新增引导进度指示器组件，支持动态分段状态与回退动画效果。
- 新增引导重置功能，支持在设置中重新触发引导流程并提供成功/错误反馈。
- 新增启动画面视频动画，支持视频播放完成后平滑淡出过渡。
- 新增启动画面波浪特效组件，支持自定义背景颜色与实时预览。
- 新增灵动岛波浪特效组件，基于 WebGL 渲染并支持强调色动态变化。
- 新增歌词翻译功能，支持在灵动岛中同步显示原文与译文。
- 新增歌词校准设置，支持启用校准与调整延迟时间以改善同步精度。
- 新增帧率限制开关，可在性能设置中控制帧率上限。
- 新增公告视频组件，支持在公告中嵌入 B 站视频。
- 新增 ESA CDN 缓存清除脚本，支持按文件 URL 或站点域名清除缓存。
- 新增 Windows 性能监控插件，支持 CPU、内存与温度快照获取。
- 新增 Windows 蓝牙辅助插件，支持设备监控、类型推断与电池电量获取。
- 新增 Windows 亮度辅助插件，支持亮度控制与监控。
- 新增 Windows 电源辅助插件，支持电源状态监控。
- 新增 Windows WiFi 辅助插件，支持网络信息获取。
- 新增 Windows SMTC 辅助插件，支持媒体会话监控与播放控制。
- 新增 Windows 应用图标辅助插件，支持从进程名、PID 或路径提取应用图标。
- 新增 Toast 监听插件，支持通知获取与抑制功能。
- 新增跑马灯文本组件，支持溢出检测与滚动显示。
- 新增媒体播放器图标显示，在 SMTC 步骤中展示对应播放器图标。

## 体验优化

- 优化主题缓存机制与透明度设置，交互响应更灵敏。
- 优化鼠标悬停效果与过渡动画，界面交互更流畅自然。

## New Features

- Added a first-launch guide window supporting language selection, theme configuration, update source selection, whitelist setup, SMTC detection, and GitHub info display for a complete initialization flow.
- Added a guide ProcessIndicator component with dynamic segment states and regressing animation effects.
- Added a guide reset feature that allows re-triggering the guide from settings with success/error feedback.
- Added a splash screen with video playback and smooth fade-out transition after completion.
- Added a splash wave effect component with customizable background color and real-time preview.
- Added a Dynamic Island wave effect component based on WebGL rendering with dynamic accent color support.
- Added lyrics translation feature displaying synchronized original and translated lyrics in the Dynamic Island.
- Added lyrics calibration settings with enable toggle and delay adjustment for improved sync accuracy.
- Added a frame rate limit toggle in performance settings to control the frame rate cap.
- Added an announcement video component supporting embedded Bilibili videos in announcements.
- Added ESA CDN cache purge scripts supporting file URL and hostname-based cache clearing.
- Added a Windows performance monitor plugin supporting CPU, memory, and temperature snapshots.
- Added a Windows Bluetooth helper plugin with device monitoring, type inference, and battery level retrieval.
- Added a Windows brightness helper plugin supporting brightness control and monitoring.
- Added a Windows power helper plugin supporting power state monitoring.
- Added a Windows WiFi helper plugin supporting network information retrieval.
- Added a Windows SMTC helper plugin supporting media session monitoring and playback control.
- Added a Windows application icon helper plugin supporting icon extraction by process name, PID, or path.
- Added a toast listener plugin with notification retrieval and suppression capabilities.
- Added a MarqueeText component with overflow detection and scrolling text display.
- Added media player icon display in the SMTC step showing the corresponding player icon.
