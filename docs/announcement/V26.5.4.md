# eIsland V26.5.4 更新日志

> **发布日期：** `2026-05-07`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

本次版本聚焦于 **设置检索效率提升**、**Agent 模型能力扩展** 与 **自定义 API Key 连接模式增强**。同时补齐多语言内容并修复了部分界面显示细节问题。

## 新功能

- 新增设置快速导航搜索能力，支持在设置页内快速定位目标配置项。
- 新增设置项描述参与模糊检索，支持通过用途和描述信息查找配置。
- 新增设置与插件市场相关内容的搜索覆盖，常用入口检索更完整。
- 新增 Agent 对 `MiniMax` 模型的选择支持，并引入 `minimax-highspeed` 系列模型。
- 新增用户自定义 API Key 本地直连模式，满足更灵活的模型接入需求。

## 体验优化

- 优化设置项搜索算法与模糊匹配策略，搜索命中率和定位速度进一步提升。
- 优化 API Key 直连模式下的 `TraceID` 文案表达，调试与排查信息更清晰。
- 补充 `MiniMax` 提供商矢量图标，并统一相关入口视觉识别。
- 优化搜索结果卡片布局与图标展示，结果列表阅读更直观。
- 完善闹钟、搜索结果与 IMAP 等界面图标在浅色/深色模式下的自适应表现。

## 问题修复

- 修复布局预览页相册轮播卡片中媒体数量显示位置错误的问题。

## 文档更新

- 补齐设置项搜索功能与快速导航更新相关多语言内容。
- 补齐插件市场卡片相关翻译内容。
- 补充设置界面 Agent 约束说明的多语言文档与执行规范说明。

感谢大家持续反馈与支持。若你在升级后发现新问题，欢迎继续反馈，我们会尽快跟进。
