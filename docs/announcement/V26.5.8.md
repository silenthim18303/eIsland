# eIsland V26.5.8 更新日志

> **发布日期：** `2026-05-11`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

V26.5.8 聚焦工具箱「格式工厂」能力落地与稳定性提升，补齐从界面到导出链路的关键环节，并同步完成代码质量与自动化评审体验优化。

## 新功能

- 新增工具箱「格式工厂」模块，支持在全展开工具箱中直接进行图片与视频相关处理。
- 新增图片格式转换能力，支持更多输出格式与 ICO 目标尺寸选择。
- 新增视频轨道分离能力，可按需提取音轨或视频轨并导出。
- 新增视频轨道导出路径自定义能力，支持用户自行选择保存位置。

## 体验优化

- 优化格式工厂分页结构与分页面标题展示，提升工具切换时的信息清晰度。
- 优化图片元数据展示区域，补充格式、尺寸、像素、完整路径等信息并改进可读性。
- 优化图片转换交互反馈，补充加载状态与进度展示。
- 优化 IP 信息查询界面按钮与局部样式一致性。

## 问题修复

- 修复多处 `eqeqeq` 规范问题，统一严格比较写法，降低潜在行为差异。
- 修复代码质量评审工作流在特定场景下异常退出的问题，提升 PR 检查稳定性。
- 修复切换灵动岛界面后出现的界面变暗闪烁问题。
- 修复打包后翻译语言图标丢失问题。

## 文档更新

- 持续更新版本变更记录，补充本次发布涉及的功能与修复说明。
