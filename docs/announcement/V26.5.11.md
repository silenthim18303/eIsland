# eIsland V26.5.11 更新日志

> **发布日期：** `2026-05-17`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

V26.5.11 重点围绕工具箱文件压缩、快捷启动体验与稳定性完善，同时补齐自动化测试流程，提升版本交付质量与日常使用可靠性。

## 新功能

- 新增工具箱图像文件压缩能力，支持从工具界面直接执行压缩任务。
- 新增文件压缩历史任务列表，并与下载任务历史保持一致的查看与管理体验。
- 新增快捷启动编辑栏拖入快捷方式能力，配置常用启动项更高效。
- 新增快捷方式图标解析兜底策略，降低特殊快捷方式无图标的出现概率。
- 新增 MaxExpand 小游戏页签，并同步更新页面展示文案与图标。

## 体验优化

- 优化开始按钮触发后的交互流程，操作反馈更连贯。
- 将历史下载任务与历史文件压缩记录迁移到独立子页面，工具箱结构更清晰。
- 优化文件压缩工具侧边栏标题文案长度与展示效果，阅读更直观。

## 问题修复

- 修复工具箱侧边栏偶发只显示键名、不显示翻译文案的问题。
- 修复下载工具界面无法通过滚轮切换到历史任务页的问题。
- 修复任务完成后 `pending` 状态未及时清除的问题。
- 修复 STT 界面光效未完整覆盖窗口的显示问题。
- 修复自动化单元测试运行环境配置错误导致的 CI 不稳定问题。

## 文档更新

- 新增并优化 GitHub 自动化测试工作流，完善测试结果展示与日志输出可读性。
- 补齐主进程、IPC、Utils、服务与网关接口相关测试用例，提升回归验证覆盖度。
- 完成本轮架构问题与注释规范修复，提升代码维护一致性。

感谢大家持续反馈与支持。若你在升级后发现新问题，欢迎继续反馈，我们会尽快跟进。
