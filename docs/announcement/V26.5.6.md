# eIsland V26.5.6 更新日志

> **发布日期：** `2026-05-10`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

V26.5.6 聚焦工具箱能力升级与内容管理链路完善，同时优化空状态反馈体验，并补充了文档与代码规范层面的维护工作。

## 新功能

- 新增「常用软件」后端内容管理能力，支持服务端统一维护软件名称、描述、下载地址、图标与排序。
- 新增「常用软件」管理面板入口，支持在后台进行新增、编辑、启用/停用与删除。
- 新增工具箱常用软件的服务端拉取能力，前端不再依赖本地固化数据。

## 体验优化

- 优化工具箱「常用软件」空状态展示，增加加载提示与更清晰的无数据反馈。
- 空状态下新增「刷新界面」操作，便于用户快速重试拉取。
- 空状态下新增「反馈问题」入口，便于在使用场景中快速提交问题。
- 持续优化常用软件区域的布局与文案可读性，提升视觉一致性。

## 问题修复

- 修复常用软件数据字段映射不一致导致的图标显示问题。
- 修复部分场景下常用软件列表请求失败后缺少明确反馈的问题。
- 修复相关模块中的注释规范与循环规则告警，提升代码稳定性与可维护性。

## 文档更新

- 更新 README 英文内容并补充最新界面预览。
- 新增 Agent 与 Toolbox 界面截图展示，完善版本功能说明。

感谢大家持续反馈与支持。若你在升级后发现新问题，欢迎继续反馈，我们会尽快跟进。
