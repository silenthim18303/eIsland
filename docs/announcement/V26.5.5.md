# eIsland V26.5.5 更新日志

> **发布日期：** `2026-05-08`
> **Github 仓库：** `https://github.com/JNTMTMTM/eIsland`

本次版本聚焦于 **核心架构模块化重构**、**代码质量与规范治理** 以及 **总览界面休息提醒组件**。通过对灵动岛主体、独立窗口与 Agent 功能的全面拆分与标准化，显著提升了工程可维护性和运行稳定性。

## 新功能

- 新增总览面板休息提醒小组件，可实时查看各提醒事项的剩余倒计时与进度。
- 新增渲染层开发规范文档，明确翻译一致性要求与 SVG 图标深浅模式自适应策略。

## 体验优化

- 灵动岛核心逻辑全面模块化，状态协调、背景媒体、悬停交互、通知订阅、播放同步等职责拆分为独立 Hook，运行效率与响应速度进一步提升。
- 独立窗口架构同步拆分，标签同步、背景设置同步、视频播放控制等逻辑各自独立，窗口切换与配置变更更加稳定。
- Agent 对话功能拆分为独立的运行器、事件处理器、授权决策与展示态模块，复杂对话场景下的稳定性明显改善。
- Agent 语音输入模块独立拆分，音频采集、PCM 帧处理与自动滚动各司其职，语音识别体验更流畅。
- 灵动岛配置体系重新组织，背景媒体、状态配置、更新源、存储键等拆分为独立配置文件，配置管理更清晰。
- 独立窗口配置模块化，类型定义、标签页、存储键与认证逻辑分别独立管理。
- 休息提醒设置界面图标选择器支持深浅模式自适应，视觉表现更统一。
- 所有导出函数补齐完整的 JSDoc 注释，包含参数与返回值说明，代码可读性大幅提升。

## 问题修复

- 修复多个 JSX 文件中残留的未使用 React 默认导入，适配 React 17+ 新 JSX 转换规范。
- 修复休息提醒调度器中使用 `for...of` 循环不符合编码规范的问题，改为高阶函数实现。
- 修复休息提醒设置页中松散相等判断不符合严格相等规范的问题。

## 文档更新

- 新增渲染层 `AGENTS.md` 开发指引，涵盖国际化校验要求与 SVG 图标主题适配规范。
- 补齐灵动岛组件、Hook、配置模块与 Agent 状态模块的完整 JSDoc 文件头注释。
