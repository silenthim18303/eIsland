> **Release Date:** *`2026-05-31`* **GitHub Repository:** *`https://github.com/JNTMTMTM/eIsland`*

**V26.5.13 聚焦于忘记密码与重置密码完整链路建设、灵动岛各状态组件架构重构，以及大范围自动化单元测试覆盖补齐，同时修复了多项影响登录注册流程稳定性的问题。**

*V26.5.13 focuses on building the complete forgot-password and reset-password flow, restructuring the Dynamic Island state components, and significantly expanding automated unit test coverage, while fixing multiple issues that affected the stability of the login and registration process.*

## 新功能

- 新增完整的忘记密码与重置密码流程，支持邮箱验证码校验身份后修改密码。
- 新增登录界面"忘记密码"入口，可直接跳转至重置密码界面。
- 新增重置密码界面布局，包含邮箱输入、验证码发送与密码确认等完整交互。
- 将插件市场调整为壁纸市场，去除插件分页，界面定位更清晰。

## 体验优化

- 优化注册、登录与重置密码界面的控件过渡动画，交互反馈更流畅自然。
- 优化滑块验证码界面的自适应颜色策略，深色/浅色主题下表现更一致。
- 优化登录与注册界面的交互文案，提示信息更准确易懂。
- 更新壁纸市场与工具界面侧边栏文案，命名更直观。
- 重构展开、悬停、空闲、登录、重置密码等状态界面的组件结构，拆分为独立模块提升可维护性。

## 问题修复

- 修复注册和登录界面流程死锁问题，切换更顺畅。
- 修复重置密码界面返回登录界面链路失效的问题。
- 修复弹性动画关闭后登录注册界面仍然残留显示的问题。
- 修复所有不符合架构设计规范的地方，补齐全部 JSDoc 注释。
- 补齐重置密码界面多语言翻译，修复缺失的忘记密码与重置密码相关 i18n 条目。

## 文档更新

- 大范围补充自动化单元测试，覆盖主进程工具函数、IPC 处理器、AI Agent、歌词解密、天气服务、小游戏、用户资料与下载引擎等模块。
- 补齐 renderer/utils、renderer/api 与 main/ipc/agent 逻辑层单元测试。
- 补齐 karaoke provider、app IPC handler、i18n、wallpaper 与 theme 工具等模块测试。
- 补齐 main IPC handler、renderer store 配置与下载引擎配置单元测试。

## New Features

- Added a complete forgot-password and reset-password flow, supporting email verification code identity check before password change.
- Added a "Forgot Password" entry on the login screen that navigates directly to the reset-password interface.
- Added the reset-password screen layout, including full interactions for email input, verification code sending, and password confirmation.
- Renamed the plugin market to wallpaper market and removed plugin pagination for a clearer interface orientation.

## Improvements

- Optimized control transition animations on the registration, login, and reset-password screens for smoother and more natural interaction feedback.
- Optimized the adaptive color strategy of the slider captcha interface for more consistent appearance in dark/light themes.
- Optimized interaction copy on the login and registration screens for more accurate and understandable prompts.
- Updated sidebar copy in the wallpaper market and tools interface for more intuitive naming.
- Restructured the component architecture for expanded, hover, idle, login, and reset-password state screens, splitting them into independent modules for better maintainability.

## Bug Fixes

- Fixed a flow deadlock issue between the registration and login screens for smoother transitions.
- Fixed the broken navigation from the reset-password screen back to the login screen.
- Fixed an issue where the login/registration screens remained visible after the spring animation was dismissed.
- Fixed all architecture design violations and completed all JSDoc annotations.
- Completed multilingual translations for the reset-password screen and fixed missing i18n entries for forgot-password and reset-password.

## Documentation

- Completed unit tests for renderer/utils, renderer/api, and main/ipc/agent logic layers.
- Completed tests for karaoke provider, app IPC handler, i18n, wallpaper, and theme utility modules.
- Completed unit tests for main IPC handler, renderer store configuration, and download engine configuration.
