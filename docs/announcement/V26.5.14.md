> **Release Date:** *`2026-06-04`* **GitHub Repository:** *`https://github.com/JNTMTMTM/eIsland`*

**V26.5.14 新增备忘录标签管理与 Markdown 编辑、剪贴板历史导出与批量清理、全展开键盘导航等多项实用功能，同时优化了多项交互细节并全面补齐测试与代码规范。**

*V26.5.14 introduces memo tagging and Markdown editing, clipboard history export and bulk cleanup, keyboard tab navigation in max-expand layout, and more, along with multiple interaction improvements and comprehensive test and code standard fixes.*

## 新功能

- 备忘录新增批量选择与删除功能，支持多条笔记一键清理。
- 备忘录新增标签管理功能，可为笔记添加标签并按标签筛选搜索。
- 备忘录新增 Markdown 编辑与预览模式，支持富文本笔记编写。
- 剪贴板历史新增类型筛选与顶部内联筛选栏，查找内容更高效。
- 剪贴板历史新增导出功能，支持将记录保存为文件。
- 剪贴板历史新增批量清理功能，支持按范围快速清理记录。
- 全展开布局新增键盘 Tab 导航，操作更快捷。
- 灵动岛新增 Escape 键导航，按下 Esc 可快速返回。

## 体验优化

- 优化备忘录批量操作的界面布局与交互细节。
- 优化备忘录标签筛选体验，筛选响应更顺滑。
- 优化备忘录 Markdown 渲染样式与工具栏布局，编辑和阅读体验更舒适。
- 备忘录勾选状态改用专属图标替代原有对号，视觉更清晰。
- 优化剪贴板历史项目的悬停效果与动画，交互反馈更细腻。
- 剪贴板历史范围选择支持深色与浅色主题自适应。
- 优化 AI 对话界面模型卡片的可见性与过渡动画。

## 文档更新

- 新增禁止 macOS 移植的平台限制声明。
- 更新技能模板与项目配置规则。

## 问题修复

- 修复 CI 工作流中 Node.js 版本不兼容导致 i18n 检查失败的问题。
- 修复多个测试文件中的代码规范问题，消除 `any` 类型、`for...of` 循环与不规范注释。
- 补齐 SvgIcon 新增图标的测试用例，修复 `Date.now` 断言的偶发失败问题。
- 修正测试文件版权头格式以通过 CI 注释规范检查。
