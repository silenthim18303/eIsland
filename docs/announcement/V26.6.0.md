> **Release Date:** *`2026-06-09`* **GitHub Repository:** *`https://github.com/JNTMTMTM/eIsland`*

**V26.6.0 新增活动热力图、CLI 实时控制台、翻译工具标签页、URL 收藏夹导入导出等多项核心功能，同时优化了 CLI 会话状态感知与权限处理体验，并修复了多项影响稳定性的问题。**

*V26.6.0 introduces the activity heatmap, CLI live console, translation tab, URL favorites import/export, and more, along with improvements to CLI session status awareness and permission handling, plus multiple stability fixes.*

## 新功能

- 新增活动热力图功能，可直观展示每日使用活跃度，支持在 CLI 标签页与用户设置中查看。
- 新增用户设置中的登录天数与热力图显示开关，支持自定义是否展示活跃数据。
- 新增 CLI 实时控制台，支持会话监控、事件流展示与实时状态跟踪。
- 新增 CLI 灵动岛状态窗口，支持会话期间的沉浸式交互体验。
- 新增 CLI 会话权限处理按钮，可直接在事件行中进行授权决策。
- 新增 CLI 会话批量选择与删除功能，管理历史会话更便捷。
- 新增 CLI 事件卡片渐进式加载与展开详情，浏览大量事件更流畅。
- 新增 CLI 事件分页导航与筛选功能，查找特定事件更高效。
- 新增外部代理检测通知，当外部代理启动或停止时自动提醒。
- 新增代理启动通知设置，支持在设置中开启或关闭通知。
- 新增翻译工具标签页，支持多语言选择与即时翻译。
- 新增 URL 收藏夹文件夹管理、导入与导出功能。
- 新增 CLI 会话通知音效，新权限请求到达时有声音提示。
- 新增 CLI 会话自动跳转，当收到新权限通知时自动切换到 CLI 视图。
- 新增 CLI 会话歌词同步显示，支持卡拉 OK 逐字效果。
- 设置侧边栏默认进入快速导航页面，不再记忆上次标签页。

## 体验优化

- 优化 CLI 热力图数据处理与会话权限处理逻辑，交互更顺畅。
- 优化 CLI 会话事件检测与通知逻辑，状态反馈更及时准确。
- 优化 CLI 会话 ID 解析与合并逻辑，会话数据更完整可靠。
- 优化 CLI 事件处理，忽略通知类型事件以减少信息干扰。
- 优化 CLI 灵动岛状态集成与最大展开标签页检查。
- CLI 状态集成 GIF 图标，根据会话阶段提供更丰富的视觉反馈。
- 优化翻译标签页语言选择图标与下拉样式，操作更直观。
- 优化 URL 收藏夹界面布局与交互细节。
- 优化 CLI 事件卡片动画效果与文本溢出处理。

## 问题修复

- 修复 CLI 阶段处理逻辑，正确识别会话开始与子代理开始事件。
- 修复 CLI 事件处理中事件名称缺失与会话结束检测不准确的问题。
- 修复 CLI 会话状态事件摘要描述不清晰的问题。
- 修复钩子脚本载荷处理与错误处理逻辑。
- 修复通知默认状态为禁用，避免未配置时产生干扰。
- 修复外部代理停止时通知进程未清理的问题。
- 修复 TypeScript 配置中的弃用警告。
- 修复 CI 环境中 Windows 命令行转义字符问题。
- 修复翻译标签页滚动事件冒泡导致的误操作。
- 修复 i18n 配置文件中无关键条目的清理。

## 文档更新

- 为 Claude Code 组件补充文档注释。
- 新增 GSAP 官方技能文档，覆盖核心框架、性能、插件与工具等模块。
- 优化热力图单元格背景颜色的一致性。
- 优化翻译标签页文本区域样式与焦点状态。
- 优化 CLI 状态内容的对齐与内边距布局。

## New Features

- Added activity heatmap for visually displaying daily usage activity, viewable in the CLI tab and user settings.
- Added login days count and heatmap toggle in user settings for customizing activity data display.
- Added CLI live console with session monitoring, event stream display, and real-time status tracking.
- Added CLI Dynamic Island state window for immersive interaction during sessions.
- Added permission handling buttons in CLI event rows for making authorization decisions directly.
- Added bulk session selection and deletion in the CLI tab for more convenient session management.
- Added progressive loading and expandable details for CLI event cards for smoother browsing of large event lists.
- Added pagination and filtering for CLI events for more efficient event lookup.
- Added external agent detection notifications that alert when external agents start or stop.
- Added agent launch notification settings to enable or disable notifications in settings.
- Added translation tool tab with multi-language selection and instant translation.
- Added folder management, import, and export functionality for URL favorites.
- Added notification sound for CLI sessions to alert when new permission requests arrive.
- Added automatic CLI view switch when new permission notifications are received.
- Added synchronized lyrics display in CLI sessions with karaoke syllable-by-syllable effects.
- Settings sidebar now defaults to the quick navigation page instead of remembering the last tab.
